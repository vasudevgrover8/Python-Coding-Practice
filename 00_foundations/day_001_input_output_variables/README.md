# Day 001 - Python Input, Output, Variables

## Topic

Python basics: input, output, variables, type conversion, and arithmetic.

## Concepts Practiced

- Taking user input with `input()`
- Displaying results with `print()`
- Converting input with `int()` and `float()`
- Storing values in variables
- Using arithmetic operators and formulas

## Problems

1. Greet a user by name
2. Calculate age after 10 years
3. Add two numbers
4. Find the product of three numbers
5. Calculate the area of a rectangle
6. Calculate the area and perimeter of a square
7. Convert Celsius to Fahrenheit
8. Calculate total marks and percentage
9. Calculate simple interest
10. Generate a basic bill

## How to Run

Open a terminal in this folder and run any file, for example:

```bash
python 01_hello_user.py
```
